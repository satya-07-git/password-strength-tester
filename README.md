# 🔐 Password Strength Tester

Password Strength Tester is a modern, user-friendly web application that instantly analyzes the strength and security level of any password entered by the user. It provides real-time feedback on how secure a password is and offers intelligent suggestions to make it stronger.

A simple and interactive web tool that analyzes password strength in real-time and shows its security level with useful feedback.

## ✨ Features

- ⚡ Real-time password strength checking
- 📊 5-Level strength meter (Very Weak to Very Strong)
- 🌈 Color-coded visual feedback
- 🔍 Detailed analysis (length, character types, patterns)
- 💡 Smart improvement suggestions
- 🔢 Entropy score and estimated crack time
- 📱 Responsive design with Dark/Light mode

## 🛠️ Tech Stack

- HTML5, CSS3 & Vanilla JavaScript
- Tailwind CSS
- zxcvbn (for advanced strength calculation)
